\section{Example}

Set path to point to import files.

\begin{Schunk}
\begin{Sinput}
> library(sss)
> filenameSSS <- "sample.sss"
> filenameASC <- "sample.asc"
\end{Sinput}
\end{Schunk}

Import and parse the files

\begin{Schunk}
\begin{Sinput}
> sssXML <- readSSSmetadata(filenameSSS)
> sss <- parseSSSmetadata(sssXML)
> asc <- readSSSdata(filenameASC)
\end{Sinput}
\end{Schunk}

Display the results.
